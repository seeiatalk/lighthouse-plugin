## 项目介绍

利用 `lighthouse` 提供的插件功能，可以定制项目的性能检测工具。

## 使用方法

- 安装依赖

```
  yarn
```

- 启动项目

```
    yarn start
```

- 运行`lighthouse`插件脚本

```
    yarn metrics
```

运行完命令之后，会在当前目录下生成性能分析报告`lhrrport.html`
